Year 1 Java Project : 2D Game

My first project in year 1 in City University of London was creating a functional 2D game in Java. This project has taught me various things, especially many object oriented programming principles such as polymorphism and encapsulation. The main objective of this game is to collect enough coins so you can advance to the next level, while having the ability to shoot and kill enemies along the way. Once enough coins have been collected, the character can move to the next level. The boss fight levels have randomly generated fireballs spawning on top of the character, requiring the player to dodge while collecting coins. At the end of the game, you can see how many coins and kills you have amassed.

Check out the code on Github: https://github.com/mxrazzz/adbs944.github.io
